Detailed Description

template<typename DataType, typename ReturnType = DataType>
struct ezIterateBitValues< DataType, ReturnType >

Helper class to iterate over the bit values of an integer. The class can iterate over the bits of any unsigned integer type that is equal to or smaller than ezUInt64.

ezUInt64 bits = 0b1101;
for (auto bit : ezIterateBitValues(bits))
{
ezLog::Info("{}", bit); // Outputs 1, 4, 8
}
