Detailed Description

Base class for objects that require reference counting with virtual destructor.

Extends ezRefCountingImpl with a virtual destructor, making it suitable as a base class for polymorphic objects that need reference counting. Use this when you need virtual function dispatch and proper destruction through base class pointers.
