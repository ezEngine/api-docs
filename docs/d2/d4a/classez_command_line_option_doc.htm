Detailed Description

ezCommandLineOptionDoc can be used to document a command line option whose logic might be more complex than what the other option types provide.

This class is meant to be used for options that are actually queried directly through ezCommandLineUtils, but should still show up in the command line option documentation, such that the user can discover them.
