Detailed Description

Base class for components that implement 'agent steering' behavior. If, moving from point A to point B using something like a navmesh.

Member Function Documentation

◆ DeserializeComponent()

void ezAgentSteeringComponent::DeserializeComponent ( ezWorldReader &  stream)
overridevirtual

Override this to load the current state of the component from the given stream.

The active state will be automatically serialized. The 'initialized' state is not serialized, all components will be initialized after creation, even if they were already in an initialized state when they were serialized.
