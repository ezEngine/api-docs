◆ m_uiBreakablePatterns

ezUInt8 ezBreakableShard2D::m_uiBreakablePatterns = 0

If not yet broken, this mask of ezBreakablePattern bits defines which patterns can be used to break it further. Small pieces may not be broken up further, medium sized ones can only use the Cellular pattern.
