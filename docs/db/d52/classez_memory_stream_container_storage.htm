Detailed Description

template<typename CONTAINER>
class ezMemoryStreamContainerStorage< CONTAINER >

Templated implementation of ezMemoryStreamStorageInterface that adapts all standard ez containers to the interface.

Note that ezMemoryStreamReader and ezMemoryStreamWriter assume contiguous storage, so using an ezDeque for storage will not work.
