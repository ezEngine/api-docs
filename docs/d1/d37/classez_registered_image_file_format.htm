Detailed Description

Base class for a registered (globally known) ezImageFileFormat.

This is an enumerable class, so all known formats can be retrieved through the ezEnumerable interface. For example:

for (auto format = ezRegisteredImageFileFormat::GetFirstInstance(); format != nullptr; format = format->GetNextInstance()) { auto& type = format->GetFormatType(); }
