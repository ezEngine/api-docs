Detailed Description

Interface for a single image conversion step.

The actual functionality is implemented as either ezImageConversionStepLinear or ezImageConversionStepDecompressBlocks. Depending on the types on conversion advertised by GetSupportedConversions(), users of this class need to cast it to a derived type first to access the desired functionality.
