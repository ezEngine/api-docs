Detailed Description

Kaiser-windowed sinc filter - highest quality but may introduce ringing.

This filter provides the highest quality scaling with excellent preservation of detail. The Kaiser window helps reduce ringing artifacts compared to an unwindowed sinc. Use higher beta values for less ringing but more blurring.

Parameter Guidelines:

  • Beta 2-4: Less ringing, more blurring
  • Beta 4-6: Good balance (recommended range)
  • Beta 6-8: Sharp but more ringing artifacts
