Detailed Description

Indicates that the string property should allow to browse for an file (or programs) outside the project directories.

Allows to specify the title for the browse dialog and the allowed file types. Usage: EZ_MEMBER_PROPERTY("File", m_sFilePath)->AddAttributes(new ezFileBrowserAttribute("Choose a File", "*.exe")),
