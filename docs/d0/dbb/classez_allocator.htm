Detailed Description

template<typename AllocationPolicy, ezUInt32 TrackingFlags = ezMemoryTrackingFlags::Default>
class ezAllocator< AllocationPolicy, TrackingFlags >

Policy based allocator implementation of the ezAllocatorBase interface.

AllocationPolicy defines how the actual memory is allocated.
TrackingFlags defines how stats about allocations are tracked.
