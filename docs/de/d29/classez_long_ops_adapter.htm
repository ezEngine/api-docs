#include <LongOpsAdapter.h>

Detailed Description

This singleton lives in the editor process and monitors all ezSceneDocument's for components with the ezLongOpAttribute.

All such components will be automatically registered in the ezLongOpControllerManager, such that their functionality is exposed to the user.

Since this class adapts the components with the ezLongOpAttribute to the ezLongOpControllerManager, it does not have any public functionality.
