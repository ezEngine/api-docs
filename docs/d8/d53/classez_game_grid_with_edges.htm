Detailed Description

template<class CellData, class EdgeData>
class ezGameGridWithEdges< CellData, EdgeData >

An ezGameGrid that not only stores data for each grid cell, but also for shared edges between them.

Edges between cells are shared. Edges are identified by indices. For a given cell you can query the four adjacent edge indices. There are 'outer' edges around the entire grid, which are only adjacent to a single cell.
