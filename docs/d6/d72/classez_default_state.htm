using ezDefaultState::CreateStateProviderFunc = ezSharedPtr<ezDefaultStateProvider> (*)(ezObjectAccessorBase*, const ezDocumentObject*, const ezAbstractProperty*)

The functor interface for the ezDefaultStateProvider factory function.

The return value is a sharedPtr as each implementation can decide whether to provide the same instance for all objects or whether a custom instance should be created for each object to allow for state caching (e.g. prefab root information). Returning nullptr is also valid for objects / containers for which the factory has no use (e.g. prefab default state provider on an object that does not belong to a prefab). The function is called for ezDefaultObjectState usage with the pProp field left blank. For ezDefaultContainerState usage pProp will point to the container property.
