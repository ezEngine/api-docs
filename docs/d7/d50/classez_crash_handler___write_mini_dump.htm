Detailed Description

A default implementation of ezCrashHandler that tries to write a mini-dump and prints the callstack.

To use it, call ezCrashHandler::SetCrashHandler(&ezCrashHandler_WriteMiniDump::g_Instance); Do not forget to also specify the dump-file path, otherwise writing dump-files is skipped.
