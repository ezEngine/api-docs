Detailed Description

Allows for a GPU texture to be read back to the CPU. Uses the same ezGALTextureCreationDescription as a normal texture for convenience. While most of the properties may be irrelevant for this purpose, the user should not have to care about that and just request a readback texture that can read back a texture of the given description.
