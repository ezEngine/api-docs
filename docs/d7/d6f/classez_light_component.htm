float ezLightComponent::CalculateEffectiveRange ( float  fRange,
float  fIntensity 
)
static

Calculates how far a light source would shine given the specified range and intensity.

If fRange is zero, the range needed for the given intensity is returned. Otherwise the smaller value of that and fRange is returned.

◆ DeserializeComponent()

void ezLightComponent::DeserializeComponent ( ezWorldReader &  inout_stream)
overridevirtual

Override this to load the current state of the component from the given stream.

The active state will be automatically serialized. The 'initialized' state is not serialized, all components will be initialized after creation, even if they were already in an initialized state when they were serialized.
