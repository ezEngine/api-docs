Detailed Description

This class allows to query additional information about how to present properties in the property grid.

The property grid calls GetPropertyState() with the current selection of ezDocumentObject's. This triggers the ezPropertyMetaStateEvent to be broadcast, which allows for other code to determine additional information for the properties and write it into the event data.
