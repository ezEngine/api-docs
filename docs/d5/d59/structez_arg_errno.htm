Detailed Description

Many Linux APIs will fill out error on failure. This converts the error into an error code and a human-readable error message. Pass in the linux errno symbol. Be careful when printing multiple values, a function could clear errno as a side-effect so it is best to store it in a temp variable before printing a complex error message. You may have to include include <errno.h> use this.
