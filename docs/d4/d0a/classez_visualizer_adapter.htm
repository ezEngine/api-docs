Detailed Description

Base class for the editor side code that sets up a 'visualizer' for object properties.

Typically visualizers are configured with ezVisualizerAttribute's on component types. The adapter reads the attribute values and sets up the necessary code to render them in the engine. This is usually achieved by creating ezEngineGizmoHandle objects (which get automatically synchronized with the engine process). The adapter then reacts to editor side object changes and adjusts the engine side representation as needed.
